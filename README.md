# Alarm Manager PWA ⏰

A streamlined Progressive Web App (PWA) designed to manage schedules and track alarm history in real-time. Built using **Softr AI** and integrated with a cloud database to ensure seamless synchronization across devices.

## 🚀 Live Demo
**[View Live Application](https://viva93124.softr.app)**

---

## ✨ Features

### 1. Alarm Dashboard
* **Total Overview:** At-a-glance view of total alarms (10+) and historical alarm events (25+).
* **Management:** Toggle existing alarms on/off, edit settings, or add new ones.
* **Customization:** Set specific repeat days (Weekdays/Weekends), sound profiles (e.g., "Clean Beep"), and snooze durations.

### 2. Event Tracking & Logs
* **Live Event Feed:** Monitor the status of every alarm (**Scheduled, Fired, Stopped, Snoozed, or Missed**).
* **Device Info:** Logs client-side information including device type (iPhone, Pixel, Windows, Mac) and browser used.
* **Search & Filter:** Quickly find specific events by status or alarm name.

### 3. PWA Capabilities
* **Installable:** Can be added to the home screen on iOS and Android for a native app experience.
* **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.

## 📖 How to Use
1.  **Add an Alarm:** Click the **+ Add alarm** button on the home screen.
2.  **Monitor Events:** Navigate to the **Alarm Events** sidebar to see a log of triggered alarms.
3.  **Search:** Use the search bar to filter through your events by status.
